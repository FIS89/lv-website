Products

We offer two families of barcode label printing software programs to meet your needs:

LabelVision 20/20
LabelVision 20/20 is an upgrade from LabelVision 2000, which was an upgrade from LabelVision 98. It is compliant with Windows XP, Vista, Windows 7, Windows 8, Windows 8.1, and Windows 10. Never before have so many powerful features been packed into such an easy to use program. Click here for more information.


LabelVision Classic
LabelVision Classic (also known as IncrediBar) is our old, discontinued product that used to run in a combination of Windows and DOS operating systems. For new labeling applications we strongly recommend the use of LabelVision 20/20. Click here for more information.
